This project is my exercise to implement the HTTP protocol from scratch using Go and raw TCP concepts.

## Goal

- Understand how HTTP messages are structured and parsed
- Practice low-level networking fundamentals in Go
- Build protocol handling step by step instead of relying on high-level frameworks
